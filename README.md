# Blink-IT-grocery-data-analysis-2025
Designed an interactive Power BI dashboard to analyze $1.2M+ total sales across outlet types, sizes, locations, and product categories, enabling multi-dimensional business insights.

Analyzed 8,500+ items to identify high-revenue product segments (Fruits, Snacks, Household) and customer preferences using dynamic filters and slicers.

Built KPI-driven visuals (Total Sales, Avg Sales, Avg Ratings, Item Count) to evaluate outlet performance across Tier 1, Tier 2, and Tier 3 cities.

Performed outlet and category-level sales analysis (fat content, outlet size, establishment year) to uncover trends supporting inventory and expansion decisions.
